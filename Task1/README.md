Task 1: Exploratory Data Analysis (EDA)

*.Overview

In this task, I performed Exploratory Data Analysis on two datasets to understand patterns and gain useful insights.

📊 Datasets

🔹 Dataset 1: Retail Sales Data
- This dataset contains category-wise sales information.
- It helps to understand how different categories perform.

🔹 Dataset 2: Menu Nutrition Data
- This dataset contains food items with calories, fat, protein, and serving size.
- It helps to analyze nutritional values across categories.


🧹 Data Cleaning
- Removed duplicate values  
- Converted units (oz to grams, fl oz to ml)  
- Extracted numbers from text values  
- Fixed inconsistent formats  


📈 Analysis
- Created Pivot Tables  
- Calculated total and average values  
- Compared categories based on calories, fat, and protein  


📊 Visualizations
- Bar Charts → Category comparison  
- Pie Charts → Distribution of values  
- Radar Charts → Nutrient comparison


🔍 Insights

From Dataset 1:
- Sales vary across categories  
- Some categories perform better than others  

From Dataset 2:
- Breakfast has high calories and fat  
- Coffee & Tea has high protein  
- Beverages have low fat  


🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts  

📌 Conclusion
This project helped in understanding data cleaning, analysis, and visualization using Excel.
